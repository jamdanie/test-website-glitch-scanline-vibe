# Mecha UI Study

A single-page UI layout study inspired by mecha and anime design language, focused on composition, hierarchy, and subtle motion.

## Live Demo
https://jamdanie.github.io/mecha-ui-study/

## What this project explores
- Strong visual focus using a single hero element
- Layered UI with glassmorphism effects
- Vertical kanji motion using CSS and JavaScript
- Responsive scaling for desktop and mobile
- Minimal UI chrome to keep attention on the subject

## Tech
- HTML
- CSS
- JavaScript
- GitHub Pages

## Notes
This project is part of an ongoing UI experimentation series focused on layout, motion, and visual clarity.
